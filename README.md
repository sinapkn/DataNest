# ذخیره سازی اطلاعات

برنامه DataNest ساده و سریع برای ذخیره و مدیریت اطلاعات در پایگاه داده اکسل.

## ویژگی‌ها
- مدیریت مخاطبین: افزودن، ویرایش، حذف
- ذخیره خودکار در `database.xlsx`
- تاریخ شمسی/میلادی بر اساس زبان
- رابط کاربری ساده با Tkinter
- فایل اجرایی آماده (`dist/app.exe`)

## اجرا
```bash
pip install -r requirements.txt
python main.py
```

یا مستقیم فایل `dist/app.exe` را اجرا کنید (نیاز به پایتون ندارد).

## منو
- **Information → Add Information**: باز کردن پنجره مدیریت
- **Language**: تغییر زبان (فارسی/English)
